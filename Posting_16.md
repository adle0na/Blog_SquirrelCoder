###코드의 흐름제어 (if와 else 1/8)

데이터 다루는 부분이 지겨울수도 있었다고 둥가둥가 해주시네요

분기문에 대해 학습을 시작합니다

```c#
class Program
    {
        static void Main(string[] args)
        {
            int hp = 100;
            bool isDead = (hp <= 0);
            
            if(isDead)
            {
                // 괄호의 내용이 참일경우 실행 ( 지금의 경우 hp가 0인 경우 실행 )
                Console.WriteLine("You are dead!");
                // 아래와 같이 이안에는 여러가지 명령어를 실행합니다
                Console.WriteLine("You are dead!"); 
                Console.WriteLine("You are dead!"); 
                Console.WriteLine("You are dead!"); 
            }
            else if (!isDead)
            {
                Console.WriteLine("You are alive!"); 
            }
        }
    }
```

### 데이터 갖고 놀기편 (데이터 연산 6/8)

사칙연산의 기본을 복습하도록 합니다 +, -, *, /

그냥 간단한 연산말고 한줄에 여러개를 넣어보도록 해보겠습니다

```csharp
int hp = 100;

hp = 2 * 100 + 1;
```
2 곱하기 100 더하기 1이 반드시 왼쪽부터 연산하진 않습니다

실제로 곱하기 우선 순위가 덧셈보다 먼저 적용됩니다

따라서 더하기에 우선 순위를 주고 싶다면 일반적인 방법이었던 괄호를 사용합니다

100 % 3의 의미는 3으로 나눴을때 나머지를 의미 합니다

따라서 결과값은 1이 출력 됩니다

다음은 증가, 감소 연산자 입니다

```csharp
int hp = 100;

hp++;
// hp = hp + 1; 와 동일한 수식입니다
hp--;
// hp = hp - 1; 마찬가지로 동일합니다

++hp;

hp++;

// 이 둘의 차이는 전위연산, 후위연산으로 더해지는 순서가 다릅니다
// 전위는 출력하기 전에 더하고 후위는 출력 한 후에 더해줍니다
```

다음은 비교 연산자입니다 <, >, ==, !=, <=, >= 등을 사용합니다

논리식에서 비교할때 사용합니다 크기 비교는 <, >, <=, >=

같냐 다르냐는 ==, !=로 기존에 배운대로 사용합니다

연산들은 true와 false로 출력이 됩니다

그리고 &&는 And ||는 OR !는 NOT등

기초적인 내용에 대해 한번더 복습하도록 합니다

And연산 ~~이고 ~~이다 두가지 조건이 동시에 만족되야 true값을 반환합니다

둘중 하나라도 false일 경우 false값을 출력합니다

or연산입니다 ~~이거나 ~~인가요? 에서 둘중 하나만 만족해도 true값을 반환합니다

둘다 false일 경우에만 false값을 출력합니다

논리값을 거꾸로 출력하고 싶은 경우 앞에 !를 주어 반대로 출력합니다

### 데이터 갖고 놀기편 (비트 연산 7/8)

이번 과정은 비트연산입니다 강의에서도 이게 중요한가 의문을 가지셨는데

이걸 못하는 신입들이 많고 쓰일때는 중요하기 때문에 설명을 해주신다고 합니다

난이도가 있는 부분이기도 하기에 어렵다면 개념 이해만 하고 넘어가도록 합니다

먼저 시프트 연산 <<와 >>에 대해 알아봅니다

```csharp
num = num << 2;
// 이값은 num 값에서 2를 밀어서 출력합니다
// 따라서 0001 이었다면 0100인 4가되고 3을민다면 1000인 8이 됩니다
// 당연히 반대로 1000, 8에서 >> 1로 민다면 0100인 4가 출력 됩니다
```
만약 시프트 연산을 하는데 음수라면 기존대로 한칸 움직이지만 앞에 부호때문에 하나 더해줍니다

따라서 비트연산을 할때는 uint로 언사인드 값을 사용하여 부호 문제를 미연에 방지합니다

and연산은 마찬가지로 둘다 1일때만 1을 반환하고 or은 둘중 하나라도 1이면 1입니다

이제부터는 심화 과정입니다 XOR은 둘중 하나라도 1이라면 반대로 0을 출력합니다

반대로 말하면 둘다 1일때만 1을 출력합니다 벌써 헷깔리기 시작하는데

(실제 사용할때는 천천히 찾아보면서 사용하도록 해야겠습니다)

XOR의 다른 특성으로는 XOR 연산에 한번더 XOR 연산을 하면 같은 값이 나옵니다

이와 같이 id와 같은 키값은 보안상 유리해서 암호화 하여 사용합니다

```csharp
int id = 123;
int key = 401;

int a = id ^ key;
int b = a ^ key;
```
